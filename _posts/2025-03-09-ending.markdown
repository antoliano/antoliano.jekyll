# Creating Lists

You can create ordered and unordered lists in Markdown.

## Unordered List

- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
- Item 3

## Ordered List

1. First item
2. Second item
3. Third item

Lists help organize information clearly!
